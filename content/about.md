# About Us

Welcome to the Net Idea Web Demo, a showcase of modern web development practices using Symfony and Bootstrap.

## Our Mission

This demonstration project illustrates how to build a professional website using:

- **Symfony Framework**: The leading PHP framework for web applications
- **Bootstrap 5**: The most popular HTML, CSS, and JavaScript framework
- **Webpack Encore**: A simpler way to integrate Webpack into your application
- **NetIdea WebBase**: A reusable library providing common website functionality

## Features

### Backend
- Symfony 6.4 framework
- Twig templating engine
- Modular architecture with reusable bundles
- Contact form with email notifications

### Frontend
- Bootstrap 5 for responsive design
- Dark/light theme toggle
- Smooth scrolling navigation
- Mobile-first approach

### Development
- Docker and Docker Compose for easy setup
- Webpack Encore for asset management
- Hot module replacement in development
- Production-optimized builds

## Technology Stack

- **PHP** 8.2+
- **Symfony** 6.4
- **Bootstrap** 5.3
- **Webpack Encore** 4.0
- **Docker** for containerization
